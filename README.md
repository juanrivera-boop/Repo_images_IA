Portal de Generadores de Imágenes con IA

Aplicación web construida en HTML5 utilizando la librería DescartesJS, que funciona como un portal interactivo para acceder rápidamente a múltiples plataformas de generación de imágenes con inteligencia artificial desde una única interfaz.

La app integra distintos servicios de IA en un entorno unificado mediante el uso de iframes dinámicos y un sistema de botones que permite cambiar entre plataformas sin salir de la página.

Características principales

Interfaz centralizada para explorar múltiples generadores de imágenes con IA.

Navegación rápida mediante botones en la parte inferior.

Integración directa de cada plataforma dentro de la misma aplicación.

Implementación basada en DescartesJS para la gestión de la interfaz interactiva.

Cambio dinámico de portal mediante variables internas (portal).

Carga inicial automática de una de las plataformas al abrir la aplicación.
